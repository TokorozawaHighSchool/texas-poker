# Texas Poker Game

This project is a simple implementation of Texas Hold'em Poker where players can compete against an AI opponent. The game is built using HTML, CSS, and JavaScript, and is designed to be played in a web browser.

## Features

- Play Texas Hold'em Poker against an AI opponent.
- User-friendly interface with clear game state updates.
- AI logic that simulates a real player.
- Responsive design for various screen sizes.

## Project Structure

```
texas-poker
├── src
│   ├── index.html        # Entry point of the application
│   ├── styles.css       # Styles for the game UI
│   └── js
│       ├── main.js      # Main application logic
│       ├── game.js      # Game logic and flow
│       ├── deck.js      # Card deck management
│       ├── ai.js        # AI player logic
│       └── ui.js        # User interface updates
├── tests
│   ├── ai.test.js       # Unit tests for AI logic
│   └── game.test.js     # Unit tests for game logic
├── package.json          # npm configuration file
├── .gitignore            # Files and directories to ignore in Git
└── README.md             # Project documentation
```

## Setup

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/texas-poker.git
   ```
2. Navigate to the project directory:
   ```
   cd texas-poker
   ```
3. Open `src/index.html` in your web browser to start playing.
